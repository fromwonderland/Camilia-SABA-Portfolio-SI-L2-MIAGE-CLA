📚 Guide d'utilisation

🖱️ Vous pouvez directement sélectionner le fichier de votre souhait depuis l'accueil

🏠 Si un fichier est déjà ouvert, vous pouvez revenir à l'accueil en cliquant sur le bouton ACCUEIL en haut de la page

📂 Vous pouvez également utiliser les menus déroulants pour sélectionner d'abord le chapitre, puis le fichier de votre choix
